# Saeed's Standalone Apps

A collection of AI-powered and utility standalone applications.

## Apps in this Repository

### 1. Suno Master Studio (v5.6)
An advanced Electron-based application for crafting and optimizing Suno AI prompts. It features a sophisticated UI for managing styles, templates, and complex song structures.
- **Technology**: Electron, React, Vite.
- **Key Features**: Pattern-based prompting, style management, and PDF parsing.

### 2. Suno Song Maker
A lightweight Neutralinojs application designed for quick Suno AI song generation.
- **Technology**: Neutralinojs, HTML/JS.
- **Key Features**: Minimalist interface, fast prompt generation.

### 3. DocReader
A cross-platform document reader with powerful Text-to-Speech capabilities, originally built for Android and later adapted for desktop.
- **Technology**: Capacitor / Tauri.
- **Key Features**: Glassmorphic UI, multi-format support, high-quality TTS fallbacks.

### 4. RingTone Master
A specialized tool for creating and optimizing ringtones, especially for Android devices.
- **Technology**: Tauri, Svelte.
- **Key Features**: OGG optimization, small file size focus, simple cutting interface.

## How to Use

### For Developers (Source Code)
1. Clone the repository.
2. Navigate to the specific app folder in `apps/`.
3. Follow the specific build instructions in each app's directory (usually `npm install` and `npm run dev`).

### For Users (Executables)
Check the [Releases](https://github.com/saeedalsuri/Saeed-Standalone-Apps/releases) page to download the latest standalone `.exe` files for each application.
